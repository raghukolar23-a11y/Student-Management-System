# Student Management System in C++

## 📌 Project Overview

The Student Management System is a console-based C++ application designed to manage student records efficiently using file handling and menu-driven operations. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations with persistent data storage.

---

# 🚀 Features

- Add new student records
- Display all student details
- Search student by ID
- Update student information
- Delete student records
- Store data permanently using file handling
- Easy-to-use menu-driven interface

---

# 🛠️ Technologies Used

- C++
- File Handling
- Object-Oriented Programming (OOP)
- STL Vector
- String Handling
